<Partial Dialectization Rules and Examples Review>
Review target: Turkey / Turkish (Turkey)

Below are rules and examples for partially dialectizing the original sentence.
Content to review:
(1) Partial dialectization rule
(2) Partially dialectized sentence

Review requirements and notes:
1. Review the reasonableness and correctness of the partial dialectization rules and the partially dialectized sentences.
2. Note that these rules and partially dialectized sentences are intended to partially dialectize the original sentence, not fully convert it into a dialectal expression.
3. Review whether the partially dialectized sentence preserves the meaning of the original sentence.
4. If changes are needed, please edit the partial dialectization rules and the partially dialectized sentences directly in revision mode.
